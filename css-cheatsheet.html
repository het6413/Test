  • color – Sets the text color. Example: color: red;
  • background-color – Sets the background color. background-color: #f0f0f0;
  • font-size – Sets the size of the font. font-size: 16px;
  • font-family – Defines the font style. font-family: Arial, sans-serif;
  • margin – Outer space around elements. margin: 20px;
  • padding – Inner spacing inside elements. padding: 10px;
  • border – Adds borders to elements. border: 1px solid black;
  • display – Defines how elements are displayed. display: flex;
  • position – Sets element positioning. position: relative;
  • width / height – Controls element size. width: 100%; height: auto;
  • text-align – Aligns text horizontally. text-align: center;
  • justify-content – Aligns flex items horizontally. justify-content: space-around;
  • align-items – Aligns flex items vertically. align-items: center;
  • box-shadow – Adds shadow effects. box-shadow: 2px 2px 5px gray;
  • transition – Smooth changes between states. transition: all 0.3s ease;
